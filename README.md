# Мастер-класс по созданию Telegram бота

## Что будем делать на мастер-классе?
* Создадим бота в Telegram
* Познакомимся с Python и библиотекой для написания ботов
* Выгрузим код в открытый доступ

# Настройка проекта
## Структура проекта
```python
art_master_class_bot
    |--- venv/ # виртуальное окружение
    |--- bot.py # сам бот
    |--- .env # секретные данные для работы с ботом
    |--- .public_env # аналог .env, только там не хранятся данные
    |--- .gitignore # файлы, которые мы не будем выкладывать
    |--- ... # оставшиеся файлы, которые нас не особо волнуют
```

## Скачивание библиотек
Запускаем виртуальное окружение
```python
$ venv\bin\activate
```
Скачиваем необходимые библиотеки
```python
$ pip install pyTelegramBotAPI # для работы с Telegram API
$ pip install python-dotenv # для работы с секретными данными
```

## Настройка секретных данных
Файл `.env` выглядит следующим образом:
```python
BOT_TOKEN=... # здесь на месте многоточия будет стоять 
            # секретный ключ, который мы получим от BotFather
```
Файл `.public_env` выглядит следующим образом:
```python
# этот файл служит инструкцией другим программистам,
# чтобы они тоже смогли создать нужные переменные в 
# .env файле

BOT_TOKEN=PASTE_YOUR_TOKEN_HERE
```
Использование секретных данных в `bot.py`:
```python
import os
from dotenv import load_dotenv

load_dotenv() # загружаем данные из файла .env

# используем эти данные
BOT_TOKEN = os.getenv('BOT_TOKEN') 

# теперь переменная BOT_TOKEN хранит в себе ключ, который нам выдал BOT_FATHER
```

## Запуск бота
Запускаем виртуальное окружение (если оно не было запущено)
```python
$ venv\bin\activate
```
Запускаем Python код с реализацией бота
```python
$ python bot.py
```
